How You Can Help

I just had some people offer to help out with this project, asking what they could do. So I thought I'd make a list.

  1. Find anything that should be a link, and make it a link.
  2. Find anyplace that says "inaudible" with a time stamp, go listen to the show, and see if you can understand what was being said.
  3. Correction any mistakes you find. (Still waiting to see if someone notices that mistake.)
  4. Transcribe new episodes! Open an issue on GitHub, so other people know you are working on an episode and we don't have two people working on the same one. Feel free to summit parts of new transcriptions while still in progress. It's ok if you don't get all the way through a show. Just do part, and then collaborate by sending a pull request and let others help finish it.

To do these tasks, you need a free GitHub account. First, you fork the repo for this site, so you have your own personal copy. Then you can make changes to your copy. And submit those changes back to the official version as a pull request.

What's Going to Happen to These Transcripts?

Well, to start with, people can read them here. This summer, I plan to start creating a proper/bigger website for The Web Ahead, with a place for each transcript to live with the show itself. Stockpiling a bunch of transcripts is a great place to start on the new site.
